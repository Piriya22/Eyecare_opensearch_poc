# EyeCare Smart Search - OpenSearch POC

## Objective

This Proof of Concept demonstrates how OpenSearch can be used to perform
fast global searches across Eye Care patient records.

## Technology Stack

- Angular 21
- ASP.NET Core 8 Web API
- OpenSearch 3.7
- JSON Sample Data

## Features

- Global Search
- Auto Suggestions
- Highlight Search Results
- Filters
- Fast Search
